# DROP Execution Evolution Package

<p align="center"><img src="https://github.com/lakshmiDRIP/DROP/blob/master/DRIP_Logo.gif?raw=true" width="100"></p>

DROP Execution Evolution Package holds the Execution Cost Market Impact Decomposition.


## Class Components

 * [***MarketImpactComponent***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/execution/evolution/MarketImpactComponent.java)
 <i>MarketImpactComponent</i> exposes the Evolution Increment Components of the Movements exhibited by an
 Asset's Manifest Measures owing to either Stochastic or Deterministic Factors.

 * [***MarketImpactComposite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/execution/evolution/MarketImpactComposite.java)
 <i>MarketImpactComposite</i> contains the Composite Evolution Increment Components of the Movements
 exhibited by an Asset's Manifest Measures owing to the Stochastic and the Deterministic Factors.
